# 01 · 项目背景

## 1.1 主题

**唐宋诗词的时代回响——以两次大断裂为锚的文本计量分析。**

我们以中国古典诗词为素材，覆盖 **公元 712 年（盛唐）至 1279 年（南宋末）**，跨越唐、五代、两宋，构建叙事型可视化系统，用可量化的文本证据考察文学如何回应时代变迁。

## 1.2 研究立意（探索性，非预设结论）

本项目**不预设**"唐外向→宋内向"的单一结论，而是以两次最剧烈的历史断裂为锚，做**探索性分析**：

> **统领问题：中国文人在两次大断裂——安史之乱（755）、靖康之变（1127）——前后，精神世界与文学表达发生了怎样的变化？**

数据已显示这并非单调的"由外向内"：唐代外向性确在下降，**但靖康之变后，南宋豪放派（辛弃疾、陆游的家国激愤）使外向性回潮**，与姜夔、吴文英的醇雅并存——一个断裂岔出两条路。这种"反弹"正是结论的一部分，而非噪音。

## 1.3 为什么是这个跨度（712–1279）

- **覆盖完整周期**：兴起—极盛—断裂—沉潜—重构—再断裂—再分化，两次"盛极而崩"互为镜像。
- **两大断裂可对照**：755 与 1127 都是"军事崩溃 + 被迫南迁 + 文学剧变"，并置二者能看出文学回应历史的共性与差异。
- **唐诗 vs 宋词**：完成古典诗词最经典的纵向对比。
- **可验证**：每个转变都有可量化文本证据（情感、意象、地理、词频多维同步）。

> 注：原计划止于 1060（北宋仁宗），但实测语料 `ci.song.*` 为**全宋词**（含南宋），且南宋豪放回潮本身极具分析价值，故拉通至南宋末。详见 `../../memory/advice/数据处理记录.md`。

## 1.4 历史分期（六期 + 两道断裂带）

```
盛唐 ──┤755 安史之乱├── 中唐 ── 晚唐 ── 五代 ── 北宋 ──┤1127 靖康之变├── 南宋
```

| 时期 | 大致时间 | 社会状态 | 文学精神 |
|:---|:---|:---|:---|
| 盛唐 | 712–755 | 开元盛世 | 昂扬、开阔、功业向往 |
| 中唐 | 756–835 | 安史之乱后、藩镇 | 沉郁、写实、转向内省 |
| 晚唐 | 836–907 | 唐末衰乱 | 感伤、绮丽、向内 |
| 五代 | 907–960 | 分裂割据 | 词体兴盛、闺阁婉约 |
| 北宋 | 960–1127 | 统一、儒学复兴 | 理性、清雅、日常审美（内向最深） |
| 南宋 | 1127–1279 | 偏安、抗金 | 豪放回潮（家国激愤）+ 醇雅并存 |

**两个人形节点**：杜甫横跨安史之乱（盛唐→中唐），李清照横跨靖康之变（北宋→南宋）——分别是两次断裂亲历的"最亮的星"。
