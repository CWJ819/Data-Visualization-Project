## 3.2 两个地基性事实

1. **繁简割裂**：唐诗繁体、宋词/五代词简体（实测唐诗繁∶简≈5554∶30，宋词≈0∶7304）。若不统一，词频/意象对比会把**繁简差异误当时代差异**。
2. **逐首创作年不可恢复**：唐 3,663 位作者中能从 bio 解析真实生卒年者极少，故**时间轴只能到"时期"粒度**，不伪造逐年数据。

## 3.3 处理管道（四步，顺序执行）

```
ingest.py → enrich_llm.py → metrics.py → qa.py
```

| 步骤 | 职责 |
|---|---|
| `ingest.py` | 繁简归一、体裁分类、定年、去重、名气标记；**新增**：加载五代词并强制归 晚唐五代 |
| `enrich_llm.py` | DeepSeek LLM 情感富化（需 `DEEPSEEK_API_KEY`；无 key 自动跳过，不阻断管道） |
| `metrics.py` | 确定性指标聚合；自动读取 enriched.jsonl，词典法+LLM双轨情感并入输出 |
| `qa.py` | 严格JSON / 时期单调 / 繁体残留=0 / 去重 / 定年对账 / 坐标范围校验 |

**各处理细节：**

- **繁简归一**：作者名与正文统一 `opencc` 繁→简；统计基于简体 `text`，`text_raw` 保留原文。
- **定年阶梯**（不伪造年份）：
  - 唐诗：`诗人→时期`查表（高置信）→ 作者 bio 年号/世代词推时期（中置信）→「未定年」；
  - 宋词：`宋词人→北宋/南宋`查表 → bio 生年阈值 1085 → 「未定年」；
  - **五代词**：全部直接标记 `period="晚唐五代"`，置信度 high（period_source="五代词集"）；
  - 「未定年」不进时间轴，但仍进唐宋对比/词频/意象聚合。
- **体裁分类修正**：`classify_tang_form()` 歌行关键词检测已前置于排律分类，且排律判定改为要求全部行字数完全一致（原实现导致盛唐排律虚高至 36.5%，修正后为 27.4%，歌行从 2.1% 升至 9.1%）。
- **去重 / 稳定 id**：五代词使用 `wudai_<sha1[:12]>` 前缀避免与全唐诗 id 冲突。
- **名气标记**：`in_anthology`（三百首名篇）、`is_representative`（名家或名篇，五代词全部标 True）、`curated_tags`（两大断裂亲历名篇）。

## 3.4 时间轴：五期 + 两大断裂

```
盛唐 ──┤755 安史之乱├── 中唐 ── 晚唐五代 ── 北宋 ──┤1127 靖康之变├── 南宋
```

**语料总计约 79,141 首**（全唐诗 57,599 + 宋词 21,049 + 五代词 493）：

| 时期 | 数量（含五代词） | | 定年来源 | 数量 |
|---|---|---|---|---|
| 盛唐 | ~8,870 | | 诗人表 | ~33,860 |
| 中唐 | ~19,132 | | bio年号 | ~9,775 |
| 晚唐五代 | **~16,125**（+492 五代词） | | 宋词人表 | ~8,688 |
| 北宋 | ~3,680 | | bio生年 | ~3,047 |
| 南宋 | ~8,055 | | 五代词集 | ~492 |
| 未定年 | ~23,278 | | 未知 | ~23,278 |

> 以上为接入五代词后的预期值；每次 `python pipeline/ingest.py` 后以实际输出为准。

可入时间轴（datable）：**约 70%**（唐诗 75%；representative 范围接近全可定年）。

## 3.5 canonical 字段（`pipeline/output/corpus/poems.jsonl`）

每行一首，字段：

`id`（全唐诗原id / `ci_<hash>` / `wudai_<hash>`）、`dynasty`（唐/宋）、`genre_type`（诗/词）、`author`、`title`、`ci_pai`、`period`、`period_source`、`period_confidence`、`datable`、`text_raw`（原字形）、`text`（简体）、`lines`、`char_count`、`form`、`in_anthology`、`is_representative`、`curated_tags[]`、`source_file`

LLM 富化字段（独立文件 `pipeline/output/enrich/enriched.jsonl`，按 id 关联）：`极性`（积极/中性/消极）、`类型[]`、`题材_llm`、`意象_llm[]`、`创作阶段`

## 3.6 仪表盘数据契约（`dashboard/data/`）

时间轴为**有序时期**（非年份桶）。所有文件严格 JSON（`allow_nan=False`，externality 私=0 用 `null`）。

| 文件 | 关键结构 |
|---|---|
| `meta.json` | `period_order[]`、`period_counts{}`、`scope`、`events[]`、`datable_in_scope`、`llm_enriched_count` |
| `sentiment_by_period.json` | `by_period[]{period, sentiment_score, poem_count, tang_score, song_score,`**`llm_sentiment_score, llm_coverage`**`}` |
| `imagery_flow.json` | `timeline[]{period, macro_per_1k, private_per_1k, externality_index}` + 宏大/私人集 |
| `genre_by_period.json` | `by_period{period:{total, forms:{体裁:%}}}` |
| `word_freq_comparison.json` | `tang_top200/song_top200/*_per_1k/*_unique_high/common_words`（均简体） |
| `place_geo.json` | `by_period{}`、`by_dynasty{}`、`place_coords{}`；注：宋词中北宋首都多以"东京/故都"出现，汴京/汴梁计数为 0 属语料用词限制 |
| `stars.json`（繁星） | `stars[]{id,title,author,dynasty,period,sentiment_score,top_imagery[],key_line,curated_tags,`**`polarity_llm,types_llm[],theme_llm,stage_llm`**`}` |
| `rupture_755.json` / `rupture_1127.json` | `before/after{sentiment,externality_index,...}`、`sentiment_delta`、`rising_words/falling_words`、`curated_stars[]{...,`**`polarity_llm,types_llm`**`}` |

**情感指标双轨**：
- `sentiment_score`（词典法）：基于 55 个积极词 / 48 个消极词计数比，覆盖全量语料，范围 [-1,+1]，趋势方向可信，绝对值参考意义有限。
- `llm_sentiment_score`（LLM精标）：`积极→+1 / 中性→0 / 消极→-1` 均值，仅覆盖 in_anthology 及策展名篇（约 600 首）；覆盖不足 3 首的时期值为 null。

> **`--scope`**：`full`（全集）/ `representative`（名家名篇，叙事默认）/ `anthology`（仅三百首）。

## 3.7 关键发现（数据已验证，跨 scope 稳健）

- **外内比（宏大/私人意象）**：盛唐 1.79 → 中唐 1.26 → 晚唐 1.07 → 北宋 0.57 → **南宋 0.73（回升）**。唐代"由外向内"清晰，靖康之变后南宋外向性反弹（豪放回潮），`full` 与 `representative` 两范围趋势一致。
- **靖康断面**：南宋升频词含"万里/西风/家国"信号；策展星精准命中辛弃疾《永遇乐·京口北固亭》、陆游、岳飞等。
- **唐宋独有词**（繁简统一后）：唐"心/身/城/门/名"（外、实），宋"东风/江南/梅花/归去/回首"（婉约、私人）。
- **五代词的位置**：李煜《虞美人》、温庭筠《菩萨蛮》等花间词及南唐二主词纳入晚唐五代，进一步充实该时期"外向→内向"转折的文本证据。

## 3.8 已知局限

1. **词典情感是代理指标**：不如人工细读精准；趋势方向可信，绝对值仅供参考。LLM 情感精标仅覆盖选集名篇（约 600 首），覆盖面有限。
2. **未定年占比约 30%**（23,278 首，多为长尾小作者）不进时间轴；时间轴结论主要由 representative 范围支撑。
3. **选集编辑偏好**：三百首反映后世取向；保留 `full` 范围作交叉对照。
4. **唐诗 vs 宋词**对比同时含"诗/词体裁"与"时代"两重差异，解读时需注明。
5. **五代词样本偏小**（约 492 首）：花间集以闺阁题材为主，私人意象占比天然偏高；虽然样本量有限，但纳入后可作为晚唐五代词风转折的独立证据层。
6. **汴京/汴梁检索缺失**：宋词语料中北宋首都常以"东京""故都"等出现，简单字面匹配无法识别，地图上该城市计数为 0，属已知数据限制。
