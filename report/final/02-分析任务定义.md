# 02 · 分析任务定义

## 2.1 核心研究问题

> **中国文人在两次大断裂——安史之乱（755）、靖康之变（1127）——前后，精神世界与文学表达发生了怎样的变化？**

这是一个**探索性、可被数据验证**的主问题。它同时触及历史（社会断裂如何影响文学）、文学史（形式如何演化）、文化心理（心态如何变化）三层，统一收敛到"两次断裂前后的对照"这条主轴。我们**不预设**"外→内"的单一结论——数据显示唐代外向性下降、北宋最内向，**而靖康后南宋外向性回潮（豪放派）**，这种分化本身就是结论。

## 2.2 为什么值得用可视化，而非纯统计 / 机器学习

- **问题是阐释性的，而非预测性的。** 我们要回答的不是"分类准确率多高"，而是"精神世界如何随时代迁移"——这类人文结论需要专家在历史语境中解读，单一统计量无法概括。
- **核心是"多维度协同的时间叙事"。** 情感、意象、体裁、地理、词汇五个维度需要在同一时间轴上**同步比对**，可视化的多视图协调（brushing & linking）天然适配，而纯统计表难以呈现这种共变结构。
- **目标是发现 non-trivial pattern。** 探索性分析依赖交互式追问（拖动时间轴、点选意象、下钻到具体诗作），这正是可视化的 details-on-demand 优势。
- **指标是代理变量，需可追溯。** 情感分、externality 指数等都是对"精神状态"的近似度量，必须能随时下钻回原诗验证，避免把模型输出当成真相。可视化保留了这条从宏观指标到微观文本的链路。

## 2.3 六个分析子任务（X 轴 = 五个有序时期）

| 编号 | 子任务 | 看什么 | 数据 / 主图 |
|:---|:---|:---|:---|
| **T1** | 情感演化 | 盛唐高 → 中晚唐低徊 → 北宋回升 → 南宋再降 | `sentiment_by_period` / 情感温度计 + 事件带 |
| **T2** | 意象演化（核心） | externality=宏/私：唐持续下降、**南宋回弹** | `imagery_flow` / 意象河流图 |
| **T3** | 体裁演化 | 唐诗体裁 → 词的兴起 | `genre_by_period` / 堆叠面积 |
| **T4** | 地理空间 | 塞北西域 → 江南都市 → 偏安江南 | `place_geo` / 诗歌地图 |
| **T5** | 唐宋词汇对比 | 唐大物象 vs 宋雅物小空间（**繁简已统一**） | `word_freq_comparison` / 双色词云 |
| **T6** | 两大断裂断面 | 755 / 1127 前后对照、升降词、亲历名篇 | `rupture_755` / `rupture_1127` / 断面专屏 |

**T1、T2 为主线**，**T6 两大断裂为戏剧高潮**，**T4 为"Poetry GIS"亮点**。
叠加"流（全集聚合）/ 繁星（三百首名篇逐首，`stars.json`）"双层。

## 2.4 取舍（明确删减，避免摊大）

- **删除：典故来源热力图。** 需要古汉语 NER + 古籍知识图谱才能识别典故出处，工程量巨大且准确率不高，性价比低，明确放弃。
- **降级为辅助：题材分类。** 题材本身无唯一答案（如《春望》可归咏史/民生/杂感），仅作辅助维度，不作主结论。
- **删除：LLM 精确年份推断。** 易产生幻觉（给争议诗作"凭空"定到某一年），改用"创作阶段 + 置信度"。详见数据说明。
- **列为"有余力再做"（第二阶段，依赖 LLM 重做后的数据）：**
  - LDA 主题模型 → 主题河流图（让数据自己说话，减少人为预设主题的质疑）。
  - 诗人社交网络（唱和/赠答/同游）。
